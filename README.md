# FingerMusic 🎶

**FingerMusic** 是一個以「手勢」控制 MIDI 音符與參數的 Web 互動音樂專案。  
透過攝影機偵測手指位置與 pinch 動作，即可在瀏覽器中即時演奏。

---

## ✨ 功能特點 (Features)

- 以手勢控制播放 MIDI 音符（例如 pinch 觸發 note on/off）
- 可依手指位置動態調整音量或 MIDI CC
- 支援自動載入 MIDI 檔案並播放
- 完全 Web 端運作：不需安裝程式、打開瀏覽器即可使用
- 視覺化顯示手指關鍵點（如 Left[8], Right[4], Right[8]）
